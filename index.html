Space Shooter Game

Created a 2D space shooter game in Java Swing, where players control a spaceship and face off against an enemy ship in a bullet hell style. Includes multiple difficulty levels and a simple game history system.

Implemented using Java Swing for the GUI and a separate thread for game logic and rendering. Game history is managed with file-based storage, and the project emphasizes object-oriented design and modular code structure.

Developed as a solo school project, focusing on core game mechanics, user interface, and learning foundational programming concepts through practical application.

Parking Lot System

Built a parking lot management system in Java Swing that enables managers to track parking spaces, vehicle entries, and user reservations. Users can also log in to manage and view their bookings through a secure interface.

Uses Java Swing for the interface and CSV files for database management, designed for modularity and easy maintenance. The system supports multiple user roles and provides real-time updates for parking availability.

Developed as part of a team for a school project, where I focused on code structure and user login functionality, while my teammates contributed to other features and front-end design.
